# Security Policy

## Supported Versions

BlockPulse is currently in active development. Only the latest version is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of BlockPulse seriously. If you discover a security vulnerability, please do **not** open a public Issue. Instead, follow the steps below:

1.  **Email me:** Send a detailed description of the vulnerability to **[SENİN E-POSTA ADRESİN]**.
2.  **Details to include:**
    *   Description of the vulnerability.
    *   Steps to reproduce the issue.
    *   Potential impact of the vulnerability.
3.  **Response Time:** I will do my best to acknowledge your report within 48 hours and provide a fix as soon as possible.

## Security Practices

- **Zero Data Tracking:** BlockPulse does not collect or send any personal data. It only connects to the Minecraft server IP you provide.
- **Open Source:** Since the code is open-source, you can audit how we handle server queries and configuration files.
- **Dependencies:** We use well-known libraries like `mcstatus` and `customtkinter`. We recommend always keeping your Python environment updated.
